# Nomad VPN

## Цель проекта
Предоставить пользователям простой и стильный лендинг VPN-сервиса с возможностью выбора тарифа и оформления подписки.

## Основные возможности

### Лендинг с тарифами
- **Цель:** Пользователь может ознакомиться с сервисом, его преимуществами и выбрать подходящий тариф
- **Описание:** Главная страница с hero-секцией (название, слоган, CTA), блоком преимуществ сервиса (скорость, безопасность, отсутствие логов, количество серверов), и блоком тарифов с карточками (демо за 1 ₽ и полный за 199 ₽) с ценами и характеристиками
- **Критерии приёмки:**
  - Hero-секция отображается с названием сервиса, кратким описанием и кнопкой призыва к действию
  - Блок преимуществ содержит 3-4 ключевые характеристики с иконками
  - Блок тарифов содержит 2 карточки с названием, ценой и списком возможностей
  - Все секции адаптивны под мобильные устройства

### Страница политики конфиденциальности
- **Цель:** Пользователь может ознакомиться с политикой конфиденциальности сервиса
- **Описание:** Отдельная страница с текстом политики конфиденциальности, доступная по ссылке в подвале сайта. Содержит разделы: какие данные собираются, как используются, как хранятся, права пользователя. Оформлена в едином стиле киберпанк/неон.
- **Критерии приёмки:**
  - Страница доступна по ссылке «Политика конфиденциальности» в футере
  - Текст политики структурирован по разделам
  - Страница стилизована под общий дизайн (тёмный фон, неоновые акценты)
  - Есть кнопка «На главную» для возврата

### SEO-статьи
- **Цель:** Пользователь может узнать полезную информацию о VPN, IP-адресах, DNS и шифровании
- **Описание:** 5 отдельных страниц с образовательными статьями: /blog/vpn, /blog/ip-address, /blog/dns, /blog/encryption, /blog/split-tunneling. Каждая статья содержит понятное объяснение темы, оформлена в едином стиле киберпанк/неон. Ссылки на статьи размещены в футере сайта.
- **Критерии приёмки:**
  - Каждая статья доступна по своему URL
  - Текст статей написан понятным языком, структурирован
  - Статьи стилизованы под общий дизайн (тёмный фон, неоновые акценты)
  - Есть кнопка «На главную» для возврата
  - Ссылки на статьи отображаются в футере

## Дополнительные заметки
- Дизайн в стиле киберпанк/неон: тёмный фон, неоновые акценты (синий, фиолетовый, зелёный), светящиеся элементы
- Только лендинг, без личного кабинета, регистрации и оплаты в первой версии
- Статические данные, без бэкенда и базы данных
